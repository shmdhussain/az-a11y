Simple Submit Button Tag

This is just a simple valid Submit Button

Zen Coding Syntax

  1. Copy & Paste the below code in the NotePad++ and Press the Expand Abbreviation Shortcut like (CTRL + ALT + Z or any which you set) then you will see the expansion of Web accessibility Compliant Tag
  2. Manually Paste and Go to "Plugins" in Menubar in NP++ >> "ZenCoding-Python" >>> "Expand Abbreviation"
azbuttonsubmit

Code:

	
<input type="submit" name="mysubmitbuton" value="Submit Button">

Few Points to Note:

  1. Value attribute (here "Submit Button") is read by screen reader. It should not be left empty.
  2. It should be present inside the <form> element.
  3. Avoid giving "name" attribute value as "submit" as it creates confusion in submitting form using javascript.
