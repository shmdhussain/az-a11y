Simple Abbreviation Tag

This is just a simple valid Screen Reader checked abbreviation tag

Zen Coding Syntax

  1. Copy & Paste the below code in the NotePad++ and Press the Expand Abbreviation Shortcut like (CTRL + ALT + Z or any which you set) then you will see the expansion of Web accessibility Compliant Tag
  2. Manually Paste and Go to "Plugins" in Menubar in NP++ >> "ZenCoding-Python" >>> "Expand Abbreviation"
azabbr

Code:

	
<abbr title="British Broadcast Corporation">BBC</abbr>

Few Points to Note:

  1. Abbreviation Tag is used to include the short form of particular entity like BBC in place of British Broadcast Corporation
