Simple Table Tag Type-1

This is just a simple valid Screen Reader checked table tag

Zen Coding Syntax

  1. Copy & Paste the below code in the NotePad++ and Press the Expand Abbreviation Shortcut like (CTRL + ALT + Z or any which you set) then you will see the expansion of Web accessibility Compliant Tag
  2. Manually Paste and Go to "Plugins" in Menubar in NP++ >> "ZenCoding-Python" >>> "Expand Abbreviation"
aztable1

Code:

	
<table>
	<caption>Cups of coffee consumed</caption>
	<thead>
		<tr>
			<th id="header1">Name</th>
			<th id="header2">Cups</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td headers="header1">Jack</td>
			<td headers="header2">10</td>
		</tr>
	</tbody>
</table

Few Points to Note:

  1. ID value of <th> tag and headers attribute value of the corresponding <td> should match.
  2. Include <caption> tag just after creating the <th> tag to mention the title of the table.

Simple Table Tag Type-2

This is just a simple valid Screen Reader checked table tag

Zen Coding Syntax

  1. Copy & Paste the below code in the NotePad++ and Press the Expand Abbreviation Shortcut like (CTRL + ALT + Z or any which you set) then you will see the expansion of Web accessibility Compliant Tag
  2. Manually Paste and Go to "Plugins" in Menubar in NP++ >> "ZenCoding-Python" >>> "Expand Abbreviation"
aztable2

Code:

	
<table>
	<caption>Student List</caption>
	<tr>
		<th scope="col">Name</th>
		<th scope="col">Age</th>
		<th scope="col">Birthday</th>
	</tr>
	<tr>
		<th scope="row">Jackie</th>
		<td>5</td>
		<td>April 5</td>
	</tr>

	<tr>
		<th scope="row">Beth</th>
		<td>8</td>
		<td>January 14</td>
	</tr>
</table>

Few Points to Note:

  1. Use scope attribute value "row" and "col" judiciously on the table header cells and table value cells.
  2. Include <caption> tag just after creating the <th> tag to mention the title of the table.
