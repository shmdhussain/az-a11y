Simple Input Textbox Type 1

This is just a simple valid input type Textbox

Zen Coding Syntax

  1. Copy & Paste the below code in the NotePad++ and Press the Expand Abbreviation Shortcut like (CTRL + ALT + Z or any which you set) then you will see the expansion of Web accessibility Compliant Tag
  2. Manually Paste and Go to "Plugins" in Menubar in NP++ >> "ZenCoding-Python" >>> "Expand Abbreviation"
azinput-text1

Code:

	
<label for="fn">Name: </label> <input id="fn" type="text" name="Firstname" value="">

Few Points to Note:

  1. This type have a explicit <label> tag.
  2. For attribute of label tag and Id attribute of input tag should match

Simple Input Textbox Type 2

This is just a simple valid input type checkbox

Zen Coding Syntax

  1. Copy & Paste the below code in the NotePad++ and Press the Expand Abbreviation Shortcut like (CTRL + ALT + Z or any which you set) then you will see the expansion of Web accessibility Compliant Tag
  2. Manually Paste and Go to "Plugins" in Menubar in NP++ >> "ZenCoding-Python" >>> "Expand Abbreviation"
azinput-text2

Code:

	
<input type="text" title="site search" name="query" id="q" value="">

Few Points to Note:

  1. This type does not have a explicit <label> tag, but the label is implicitly specified in the title attribute of input tag
  2. Title will read out the value (need not be equal to Id attribute of the input tag)
