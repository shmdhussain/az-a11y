Zen/Emmet Coding

Emmet(Previously Zen coding) Coding is a shortcut, error free, productive and easy way of coding HTMLs, CSS (tags and code snippets), particularly remove the need to remember the code snippets we used everyday

How to Use?

  1. Copy & Paste the zen code format in the NotePad++ and Press the Expand Abbreviation Shortcut like (CTRL + ALT + Z or any key combo you set) then you will see the expansion of HTML Tag/Snippet
  2. Zen Coding has it's own pattern to expand the normal HTML tags and snippets.
  3. Example on giving the zen coding short cut pattern as in the greenbox below, will give the expanded actual HTML coding as in below code block.
  4. Try the below General Zen Coding Examples in notepad++ or any IDE where Zen Coding is supported.

What we have done With Zen coding?

  1. For Screen Readers(eg:- NVDA, VoiceOver in iOS, TalkBack in Android and Windows Phone Screen reader) and other Assistive technologies, we have collected the basic HTML tags
  2. These basic tags are assigned a short cut in zen coding and one can use the short cut which we have feed to expand the Web Accessibility compliant tag.
  3. These short cuts can seen and searched in Nav bar.
  4. See the examples in Examples for az-a11y section below
