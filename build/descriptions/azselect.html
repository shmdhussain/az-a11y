Simple Select Tag

This is just a simple valid select dropdown

Zen Coding Syntax

  1. Copy & Paste the below code in the NotePad++ and Press the Expand Abbreviation Shortcut like (CTRL + ALT + Z or any which you set) then you will see the expansion of Web accessibility Compliant Tag
  2. Manually Paste and Go to "Plugins" in Menubar in NP++ >> "ZenCoding-Python" >>> "Expand Abbreviation"
azselect

Code:

	
<select id="favsports" name="select">
	<option value="1">Cricket</option>
	<option value="2">Football</option>
	<option value="3">Badminton</option>
	<option value="4">Tennis</option>
	<option value="5">Basketball</option>
	<option value="6">Rugby</option>
	<option value="7">Squash</option>
</select>

Few Points to Note:

  1. For attribute of label tag and Id attribute of select tag should match
  2. <optgroup> can also be used, but check for browser compatibility, not all browsers supported <optgroup>.
