Simple Anchor Tag

This is just a simple valid Screen Reader checked anchor tag

Zen Coding Syntax

  1. Copy & Paste the below code in the NotePad++ and Press the Expand Abbreviation Shortcut like (CTRL + ALT + Z or any which you set) then you will see the expansion of Web accessibility Compliant Tag
  2. Manually Paste and Go to "Plugins" in Menubar in NP++ >> "ZenCoding-Python" >>> "Expand Abbreviation"
aza

Code:

	
<a id="optionalID" class="optionalClass"  title="optionalTitle" href="Product.html" >Product Page</a>

Few Points to Note:

  1. href atribute is must and id, class, title attribute are optional
  2. In angular application or you are not having any navigation link, use javascript:void(0) in href attribute value
